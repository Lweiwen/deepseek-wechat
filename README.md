## 项目描述

* 通过对接DeepSeek API与微信接口实现的智能聊天机器人，支持自动化消息响应。
* Description: A WeChat chatbot integrated with DeepSeek's API for automated message replies.

## 核心功能

* 微信消息实时监听
* DeepSeek多轮对话接口调用
* 上下文敏感型回复生成
* 异常流量熔断机制
